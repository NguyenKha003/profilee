Git cơ bản cho sinh viên: đủ dùng để làm đồ án nhóm

Git giúp bạn lưu lịch sử code và làm việc nhóm an toàn. Nếu đồ án nhóm mà không dùng Git, tới ngày cuối rất dễ "vỡ trận".

1) 5 lệnh quan trọng nhất

  • git status: xem file thay đổi
  • git add .: đưa file vào stage
  • git commit -m "...": lưu một mốc
  • git push: đẩy lên GitHub
  • git pull: kéo code mới về

2) Làm nhóm đúng cách: branch + PR

  • Mỗi tính năng làm 1 branch: feature/login, feature/products
  • Push branch lên GitHub
  • Tạo Pull Request (PR) vào main
  • Review rồi merge

3) Quy ước commit message

  • feat: thêm tính năng
  • fix: sửa lỗi
  • docs: sửa README/tài liệu

4) Tránh conflict

  • Luôn pull trước khi code
  • Không sửa cùng 1 file lớn nếu không phân chia
  • Commit nhỏ và push thường xuyên
