Java cơ bản: kiểu dữ liệu, biến và hằng số

Đây là phần nền tảng mà đi làm bài tập Java (và làm đồ án) bạn sẽ gặp hoài: kiểu dữ liệu, biến và hằng số. Nắm chắc 3 thứ này sẽ giúp bạn tránh nhiều lỗi “ngớ ngẩn” khi code.

1) Kiểu dữ liệu nguyên thủy (primitive)

  • int, long: số nguyên
  • double, float: số thực
  • boolean: đúng/sai
  • char: ký tự

2) Kiểu tham chiếu (reference)

Ví dụ phổ biến: String, ArrayList, object do bạn tự tạo. Khác với primitive, reference có thể mang giá trị null.

3) Khai báo biến

int age = 20;
double gpa = 3.2;
String name = "Nguyễn Kha";

4) Hằng số với final

final double PI = 3.14159;
// PI = 3.14; // lỗi: không được gán lại

Mẹo: hằng số thường đặt tên IN HOA để dễ nhận biết.

5) Lỗi sinh viên hay gặp

  • Nhầm String so sánh bằng == thay vì equals().
  • Quên ép kiểu khi chia số nguyên: 5/2 ra 2 (không phải 2.5).
  • Gặp NullPointerException vì gọi method trên biến đang null.
