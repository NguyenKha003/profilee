REST API là gì? Hiểu nhanh để làm web và đồ án

REST API là cách mà frontend (trình duyệt) và backend (server) giao tiếp với nhau qua HTTP. Nói đơn giản: bạn gọi một đường dẫn (endpoint) và nhận dữ liệu (thường là JSON).

1) Endpoint là gì?

Endpoint là URL đại diện cho tài nguyên. Ví dụ:

  • /api/users: danh sách người dùng
  • /api/products: danh sách sản phẩm

2) CRUD với REST

  • GET /api/products → lấy danh sách
  • GET /api/products/1 → lấy chi tiết
  • POST /api/products → tạo mới
  • PUT /api/products/1 → cập nhật
  • DELETE /api/products/1 → xoá

3) Status code quan trọng

  • 200: OK
  • 201: Created (tạo thành công)
  • 400: Bad Request (dữ liệu gửi lên sai)
  • 401: Unauthorized (chưa đăng nhập)
  • 403: Forbidden (không có quyền)
  • 404: Not Found
  • 500: Server Error

4) Ví dụ gọi API bằng JavaScript

async function loadProducts() {
  const res = await fetch("/api/products");
  if (!res.ok) throw new Error("HTTP " + res.status);
  return await res.json();
}

5) Lỗi thường gặp

  • Gọi sai URL/endpoint → 404
  • Quên parse JSON → không đọc được dữ liệu
  • Không xử lý res.ok → lỗi khó debug
