JavaScript DOM & Event: thao tác HTML và bắt sự kiện (ví dụ dễ hiểu)

Làm web mà không biết DOM và event thì giống như có HTML/CSS nhưng không “điều khiển” được gì. Bài này tóm tắt các thao tác hay dùng nhất.

1) DOM là gì?

DOM (Document Object Model) là cách trình duyệt biểu diễn HTML thành cây đối tượng. Bạn dùng JS để chọn phần tử và thay đổi nội dung/style.

2) Chọn phần tử

const title = document.querySelector("h1");
const btn = document.querySelector("#btnSave");
const items = document.querySelectorAll(".item");

3) Bắt sự kiện với addEventListener

btn.addEventListener("click", () => {
  title.textContent = "Đã bấm nút!";
});

4) Lỗi hay gặp

  • Script chạy trước khi HTML render xong → đặt script cuối body hoặc dùng defer.
  • querySelector trả về null vì selector sai.
  • Nhầm onclick inline với addEventListener (nên ưu tiên addEventListener).
