Java Collections: ArrayList vs HashMap (khi nào dùng cái nào?)

Khi làm bài tập Java hoặc làm đồ án CRUD, bạn sẽ gặp 2 cấu trúc rất hay dùng: ArrayList và HashMap. Bài này giúp bạn phân biệt và chọn đúng cái cần dùng.

1) ArrayList: danh sách theo thứ tự

  • Dữ liệu có thứ tự (index 0..n-1)
  • Duyệt danh sách rất trực quan
  • Tìm theo “id” thì phải tự loop
ArrayList names = new ArrayList<>();
names.add("Kha");
names.add("An");

2) HashMap: tra cứu theo key

  • Dùng cặp key-value
  • Tra cứu theo key nhanh
  • Hợp khi bạn có “id” rõ ràng
HashMap score = new HashMap<>();
score.put("kha", 10);
int s = score.get("kha");

3) Khi nào dùng cái nào?

  • ArrayList: danh sách hiển thị (list sản phẩm, list sinh viên…), cần sắp xếp, cần duyệt tuần tự.
  • HashMap: cần tra nhanh theo id/key (tìm user theo username, tìm sản phẩm theo mã).

4) Lỗi hay gặp

  • Quên import: import java.util.*;
  • Nhầm generic: ArrayList<int> (sai) phải dùng ArrayList<Integer>.
  • map.get(key) trả về null nếu key không tồn tại.
