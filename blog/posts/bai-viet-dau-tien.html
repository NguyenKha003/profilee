Lập trình hướng đối tượng (OOP) là gì?

OOP là nền tảng quan trọng trong Công nghệ Phần mềm, giúp chương trình dễ bảo trì và mở rộng. Trong các môn Java và C#, mình áp dụng OOP để xây dựng các ứng dụng thực tế.

4 tính chất cơ bản

  • Đóng gói: gom dữ liệu & hành vi vào một lớp
  • Kế thừa: tái sử dụng và mở rộng lớp cha
  • Đa hình: cùng một interface nhưng nhiều cách triển khai
  • Trừu tượng: ẩn chi tiết, chỉ lộ ra những gì cần thiết

Một đoạn Java ví dụ

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello from Nguyen Kha!");
    }
}

Khi làm đồ án, OOP giúp mình tách lớp rõ ràng (model/service/controller), giảm lặp code và dễ mở rộng tính năng.
